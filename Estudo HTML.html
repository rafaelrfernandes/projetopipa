Tags de texto e suas funções

Tag Características Função Observação
\div\ Elemento em bloco Fazer a divisão de textos distintos Dentro desta TAG é possível ter quantas TAGS quiser pela característica de flexibilidade
\hx (do 1 ao 6)\ Elemento em bloco, ou seja, se comporta igual a TAG div Serve para colocar título
\title\ Utilizada como título da aba do navegador
\p\ Aplicação de textos com parágrafos atuam por blocos.
\span\ Inline Basicamente a mesma função do parágrafo, porém tag é repetida na linha de código abaixo. Possível uso quando se tratar de informações sensíveis em um texto, as quais exigem mais atenção, logo maior facilidade de visualização.
\b\ Inline Negrito
\u\ Inline Sublinhado
\i\ inline Itálico
\br\ inline Separar linhas de código com a TAG \span\ ou espaçamento entre linhas

Tags de imagem, de link, atributos e suas funções

Tag Características Função Observação
\img\ scr="" inline (adicionar TAG \br\ para ter quebra de linha) Adicionar imagens Necessário colocar o atributo scr="" para informar a fonte
\a href="endereço">Link!<\a Inline Adicionar links
\target="_blank" Inline Abre o link em nova janela Atributo
\table\ Abrir estrutura da tabela Estrutura
\thead\ Título, primeira linha e colunas da tabela Utilizar para definir variáveis Necessário utilizar \tr\ e depois \th\
\tbody\ Preencher as linhas colunas da tabela Utilizar para preencher a tabela Necessário utilizar \tr\ e depois \td\
align="" Inline Definir alinhamento atributo
width="" Inline Definir largura atributo
border"" Inline Definir borda atributo
height="" Inline Definir altura atributo

\form\ Estrutura do formulário Fazer a base do formulário

\textarea\ Ajustável Inserir caixa de texto
\button\ Adicionar botão Inserir o texto desejado. Ex.: Enviar

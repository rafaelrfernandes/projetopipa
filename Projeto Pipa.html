Tag 'span' tem basicamente a mesma função do parágrafo, porém tag é repetida abaixo. A partir daqui a tag \span\ está na linha abaixo (pré definida como tag inline, que significa que abre e fecha na mesma linha), diferente do parágrafo.

Ainda no estudo sobre a tag \span\ é possível ter o mesmo efeito com a tag de quebra de linha \br\ entre linhas da tag span (tag inline). Para gravar: tags inline (ex. chraset) podem ser fechadas com a barra no final ou sem ela, melhor utilizar para fins de organização
